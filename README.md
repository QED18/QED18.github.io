## Key Features

- **Simple Syntax**
- Supports _italic_, **bold**, and ***bold italic*** text
- Ability to create:
  - Lists
  - Tables
  - Links
  - Images

### Example Syntax

1. **Headers**  
   Use `#` for headers. Example:  
   `# Header 1`  
   `## Header 2`

2. **Lists**  
   - Unordered: Use `-` or `*`  
   - Ordered: Use numbers

3. **Links**  
   `[OpenAI](https://openai.com)`

4. **Images**  
   `![Image Alt Text](image-url.jpg)`
